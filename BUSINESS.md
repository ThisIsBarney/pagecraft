# PageCraft 商业化路线图

## 核心目标
做一个能持续盈利的 Notion 建站工具，月收入目标 $1,000+

## 市场验证（已做）
- ✅ 需求存在：Notion 用户想拥有精美网站
- ✅ 竞品验证：Super.so 收费 $12-16/月，有付费用户
- ✅ 技术可行：已打通 Notion API + 部署

## 差异化定位
**"中国用户的 Super.so 平替"**
- 价格更低（$6/月 vs $12/月）
- 国内访问更快
- 支持中文优化

## 商业化策略

### 免费版（获客）
- 1个页面
- 基础模板
- 子域名（pagecraft.io/xxx）
- 带 "Made with PageCraft" 水印

### Pro版（$6/月 或 $48/年）
- 无限页面
- 所有高级模板
- 自定义域名
- 去水印
- 密码保护
- 分析统计

### 团队版（$12/月）
- 多成员协作
- 团队模板库
- API 访问
- 优先支持

## 关键成功因素
1. **模板质量** - 必须好看，这是核心竞争力
2. **SEO友好** - 能被 Google 收录
3. **加载速度** - 国内访问要快
4. **易用性** - 3分钟完成建站

## 下一步优先级
1. 做3-5个精美模板（证明差异化）
2. 自定义域名（核心付费功能）
3. 用户系统（支撑付费）
4. 支付集成（收钱）
