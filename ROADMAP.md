# PageCraft MVP 路线图

## 目标
把 Notion 页面转换成精美的个人主页/作品集网站

## MVP 功能（第1阶段）
- [ ] 用户连接 Notion 集成（OAuth）
- [ ] 选择要展示的 Notion 页面
- [ ] 基础模板渲染（1个极简模板）
- [ ] 生成可访问的公开链接
- [ ] 基础移动端适配

## 第2阶段
- [ ] 自定义域名支持
- [ ] 2-3个新模板（设计师、开发者、摄影师）
- [ ] SEO 优化
- [ ] 页面加载速度优化

## 第3阶段
- [ ] 付费墙（自定义域名、高级模板）
- [ ] 分析统计（访问量）
- [ ] 多页面支持（整个网站而不仅是单页）

## 技术栈
- Next.js 14 (App Router)
- Notion API
- Tailwind CSS
- Vercel 部署
- 数据存储: Vercel KV / Upstash Redis
